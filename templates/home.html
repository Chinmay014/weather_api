Weather Data API

URL format: http://127.0.0.1:5000/api/v1/station/date

URL Example: http://127.0.0.1:5000/api/v1/10/1989-02-16

For displaying data for one particular station. http://127.0.0.1:5000/api/v1/10

For displaying data for one particular station on a particular year(eg.,1988): http://127.0.0.1:5000/api/v1/yearly/10/1988
